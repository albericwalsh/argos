# ARGOS #
## Installation ##
```bash
python -m venv argos_env
argos_env\Scripts\Activate.ps1
pip install -r requirements.txt
```
## Usage ##
### Lancement de l'application CLI only ###
```bash
python argos.py 
```
### Lancement de l'application avec l'interface web ###
```bash
python argos.py --web
```
