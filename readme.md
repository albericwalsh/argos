# ARGOS #
## Installation ##
```bash
python -m venv argos_env
source argos_env/bin/activate  # On Windows: argos_env\Scripts\activate
pip install -r requirements.txt
```
## Usage ##
### Lancement de l'application CLI only ###
```bash
python argos.py 
```
### Lancement de l'application avec l'interface web ###
```bash
python argos.py --web
```
