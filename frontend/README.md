# Argos Frontend

Frontend statique minimal pour Argos (thème noir / rouge).

- Fichiers créés:
  - `index.html`
  - `css/style.css`
  - `js/app.js`

Pour tester localement, ouvrir `index.html` dans le navigateur. Pour intégrer au serveur WebUI, copier le contenu dans le dossier `Argos/WebUI/static` ou servir via le serveur existant.
